# Task 1: Web Security Vulnerability Assessment
Internship ID: FUTURE_CS_01  
Target Site: http://testphp.vulnweb.com

# Project Overview
This project involved a non-intrusive, professional security audit of the Acunetix test environment. The goal was to identify critical security weaknesses and provide business-friendly remediation steps.

## 🛠️ Tools Used
Nmap (HackerTarget): Port and service discovery.
Browser DevTools: Header and cookie analysis.
SecurityHeaders.com: Passive configuration audit.
Canva: Professional report design.

#  Findings Summary
Critical: Outdated PHP 5.6.40 (End-of-Life).
High: Lack of SSL/HTTPS encryption (Port 443 Filtered).
Medium: Security Header Grade "F" & Insecure Cookie Flags.
